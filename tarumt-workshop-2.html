What is
Microservices?

  • What is microservices?
  • Microservices - also known as the microservice architecture - is an architectural style that structures an application as a collection of services that are
    1. Independently deployable
    2. Loosely coupled
    3. Organized around business capabilities
    4. Owned by a small team
    5. Highly maintainable and testable
  • In this workshop, you will learn how to build an backend web server by using ASP.NET Core.
  • Agenda:
    1. Welcoming
    2. Building Microservices backend using ASP.NET Core
    3. Implement Authentication & Authorization using OpenID Protocol based on OAuth Specifications
    4. QnA Session
    5. Photo Taking Session
    6. Closing

  • 🗓️ Date: 12/2/2023 (Sun) ⏰ Time: 8:00pm - 10:00pm 📍 Venue: Google Meet

Prerequisite

  1. Install .NET 7.0 SDK.
  2. Install Visual Studio 2022.
  3. Install Ubuntu on WSL2.
  4. Install MariaDB on Ubuntu.
